# go-mongo-rest

Example of how to make REST endpoints with Go and Mongodb.

## How to run

Copy/paste this into your terminal:

```
git clone git@github.com:jakecoffman/go-mongo-rest.git
cd go-mongo-rest
go run cmd/server/server.go
```
